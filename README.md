# 🎨 NpureStudio
日本語プロンプトをその場で画像化する、オンデバイスAI生成プラットフォーム。

本プロジェクトは iPhone の NPU（Safari WebGPU）を活用し、ユーザの日本語入力を起点にローカルで高速な画像生成・編集を行うことを目指します。クラウドに依存せずプライバシーを保ちながら、低レイテンシでインタラクティブな体験を提供します。

## コア方針
- 日本語をファーストクラスで扱う（日本語→英語の自動翻訳レイヤを内蔵）。
- Text-to-Image を最優先機能とし、オンデバイスでの高速生成を実現する。
- 高負荷な処理は非同期／サーバフォールバック可能にし、常に UX を優先する。

## 主要機能（優先順位）
1. 日本語 Text-to-Image（最優先）
	- 日本語プロンプトを軽量LMで英訳し、WebGPU で SD 系モデルを実行して画像を生成します。
2. 生成ベースのインペイント／着せ替え（応用）
	- 生成画像や既存写真の一部を日本語で指示して差し替えます。SAM 等の座標指定は補助的に利用します。
3. マルチキャンバス編集
	- 生成画像をキャンバス上で重ね合わせ・配置・微調整できる UI。

## クイックスタート（ローカル）
依存をインストールし、ローカルサーバで動作確認します。

```bash
npm install
npx http-server . -p 8080
```

必要に応じて HTTPS プロキシを別コンソールで起動します:

```bash
npx local-ssl-proxy --source 9000 --target 8080
```

ブラウザでアプリを開き、DevTools のコンソールで初期化ログを確認してください。

## 技術スタック（抜粋）
- Runtime: WebGPU（Safari）
- Engine: Transformers.js v3（WebGPU/WASM バックエンド）
- Translation: 軽量 T5 系モデル（日本語→英語）
- Generation: ONNX 化した SD 系モデル（量子化必須）

詳細は `docs/概要設計.md` と `docs/詳細設計.md` を参照してください。
