# 🎨 NpureStudio
**iPhone 17 ProのNPUを解放する、日本語対応オンデバイスAI生成プラットフォーム**

## ✨ コンセプト
「言葉が、その場で形になる。」
クラウドに頼らず、iPhone内部のNPU（Safari WebGPU）をフル活用し、日本語プロンプトから爆速で画像を生成します。

- **Native Japanese Support:** 翻訳を意識せず、日本語で自由に指示。
- **Pure NPU Acceleration:** Safari 26+ の WebGPU 経由で iPhone の演算能力を限界まで引き出す。
- **Privacy First:** 全ての推論はデバイス内で完結。画像が外に出ることはありません。
