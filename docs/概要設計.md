# NpureStudio 概要設計書

## 1. プロジェクト概要
iPhoneのNPU（Neural Processing Unit）とWebGPU技術を活用し、Safariブラウザ上で完結する「オンデバイスAI画像生成・編集プラットフォーム」を構築する。

## 2. 解決したい課題
- 既存アプリ（Draw Things等）における「単一キャンバス」「画像入力数の制限」といったUIの不自由さを解消する。
- クラウドサーバーを介さず、プライバシーを保護しながら爆速でAI編集を行う。

## 3. 主要機能 (Core Features)
### A. 自動着せ替え（Virtual Try-On）
- **入力:** 人物写真、衣服写真の2点。
- **処理:** 1. AIによる人物・衣服の自動セグメンテーション（マスク生成）。
    2. IP-Adapter等を用いた衣服の特徴抽出。
    3. インペイント技術による衣服の差し替え。
### B. 汎用画像生成・編集
- **Text-to-Image:** プロンプトからの画像生成（SDXL Turbo等）。
- **Image-to-Image:** 写真のスタイル転送、背景チェンジ。
- **Object Removal:** 不要なものの削除。

## 4. 技術スタック
- **Runtime:** WebGPU (Safari 26+)
- **Engine:** Transformers.js v3
- **Framework:** Vanilla JS or React (TBD)
- **Deployment:** GitHub Pages
