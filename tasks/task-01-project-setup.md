# Task 01: プロジェクト基本構造のセットアップ

## 概要
NpureStudio の基本的なプロジェクト構造を構築し、必要なフォルダと初期ファイルを準備する。

## 目標
- 開発環境の準備
- 基本的なファイル構造の確立
- HTML/CSS/JS の初期テンプレート作成

## 必要なファイル/コンポーネント
- `index.html`: メインHTMLファイル
- `src/`: ソースコードフォルダ
- `assets/`: 静的アセットフォルダ
- `package.json`: npm パッケージ管理

## 実装手順
1. `src/` フォルダを作成
2. `assets/` フォルダを作成
3. `index.html` を更新して基本レイアウトを追加
4. `package.json` を作成し、Transformers.js を依存関係に追加
5. 基本的なCSSとJSファイルを `src/` に配置

## テスト方法
- HTMLファイルをブラウザで開き、基本レイアウトが表示されることを確認
- コンソールにエラーがないことを確認

## 依存関係
- なし
