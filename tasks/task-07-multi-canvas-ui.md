# Task 07: マルチキャンバスUIの構築

## 概要
複数のキャンバスをサポートするUIを構築する。

## 目標
- マルチキャンバスレイアウトの実装
- キャンバス間の操作連携

## 必要なファイル/コンポーネント
- `src/ui/canvas-manager.js`: キャンバス管理モジュール
- `src/ui/multi-canvas.js`: マルチキャンバスUIコンポーネント

## 実装手順
1. `src/ui/canvas-manager.js` を作成し、キャンバス管理ロジックを実装
2. `src/ui/multi-canvas.js` を作成し、UIレイアウトを実装
3. HTML/CSS を更新してマルチキャンバスをサポート

## テスト方法
- 複数のキャンバスが正しく表示・操作できることを確認
- タッチ操作（ピンチ、ドラッグ）が機能することをテスト
